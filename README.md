Hello this is made for a game called Project Nikoli ( a game on roblox )

NPCService is a quick-easy solution to roblox npc loading which allows quick loading for custom npc's!

//-- How to use
When making an npc's logic you must:
1. add in the spawnparts and add them to the npc's spawnpart list
2. next add ( or use the template logic ) for the npc
3. and start it up ( its that simple )
